## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- cookie-session
- morgan

## Getting Started

- Fork and clone project
- Install all dependencies

```bash
npm install
```

- Run the development web server

```bash
node server/express_server.js
```

or

```bash
npm run dev
```
